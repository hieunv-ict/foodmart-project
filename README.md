# Frontend Project
## lưu ý:
- Quản lý + đóng gói bằng npm+webpack
- Chỉ làm việc với file .html, .css, .js trong folder src
## Cách chạy project
- Unzip file nén chứa project
- npm install để tải các packages
- npm run dev để chạy web
