# Frontend Project
- Quản lý + đóng gói bằng npm+webpack
- Chỉ làm việc với file .html, .css, .js trong folder src
- Cách chạy: npm install khi mới clone về để tải các packages
- Chạy command npm run dev trên CLI để chạy web
- Sau khi npm run dev, các thay đổi sẽ được cập nhật real-time
